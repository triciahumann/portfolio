Projects

Guess the Word Game

An online game created using Vanilla JavaScript and DOM Manipulation. Players guess a random word pulled from an API one letter at a time. If the player guesses all the letters correctly before they use their allotted guesses, they win!

Skills: Vanilla JavaScript, DOM Manipulation, and API’s

Tools: CodeSandbox and GitHub

Sticky Note App

My largest greenfield project to date. This is an interactive app that lets users create “sticky notes” on their computer to keep track of to-do items. CodeSandbox and the React library was used to create and manage this project.

Skills: React, ES6, JSX, and DOM Manipulation

Tools: CodeSandbox

GitHub Repo Gallery

One of my personal favorite projects to date, this is a single page website featuring a GitHub Repo Gallery. The gallery includes all of my note-worthy projects that have been shared on GitHub. Users can search for repos and view individual projects. JavaScript and API’s were used to create this project.

Skills: JaveScript and API's

Tools: CodeSandbox and GitHub

Name Tag Generator

This app is a name tag generator that can be used to create bulk name tags using functional components and ES6 modules. CodeSandbox was instrumental to the creation of this project. (Fun fact: I loved creating this project because I got to test it out regularly using the names of my most loved ones)

Skills: ES6, Functional components, React, and JSX

Tools: CodeSandbox

Unplugged Outdoor Retreat Website

A 3-page responsive website, including a Home, FAQ, and About page created from a design comp that adapts to mobile, tablet, and desktop screens. A submission form and live map are also included for the user to use and collect more information. Created by using CSS3, Flexbox, HTML5, media queries, and responsive typography. The tools used to complete this project were GitHub, Photoshop, and Visual Studio Code.

Skills: HTML5, CSS3, Flexbox, Media Queries, and Responsive typography

Tools: GitHub, Photoshop, and Visual Studio Code
