Guess the Word Game

An online game created using Vanilla JavaScript and DOM Manipulation. Players guess a random word pulled from an API one letter at a time. If the player guesses all the letters correctly before they use their allotted guesses, they win!

Skills: Vanilla JavaScript, DOM Manipulation, and API’s

Tools: CodeSandbox and GitHub
